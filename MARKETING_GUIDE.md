# VinPop Marketing Guide

## Positionnement
VinPop aide les gens à acheter du vin qu’ils aiment vraiment, sans jargon, sans snobisme, grâce à la data.

## Villain
Les experts/snobs du vin qui compliquent inutilement le choix.

## Promesse
Arrêter de gaspiller de l’argent dans des bouteilles décevantes.

## Produit
Test kit : 3 vins pour découvrir le goût du client.
Le client note Love / Like / Dislike.
VinPop recommande ensuite des box personnalisées.

## Ton
Simple, direct, anti-snob, humain, légèrement provocateur mais pas agressif.

## Cibles retargeting
1. Personnes ayant visité le site sans acheter.
2. Personnes ayant commencé le checkout.
3. Personnes ayant acheté le kit mais pas encore noté les vins.
4. Personnes ayant noté les vins mais pas acheté de box.
5. Anciens acheteurs inactifs.

## Formats à produire
- Instagram feed 1080x1350
- Story 1080x1920
- Reel script 20-30 secondes
- Meta Ads primary text
- Headline
- Description
- CTA
